# Product Overview

"Hoc Vui" is an educational quiz game for Vietnamese grade-2 students. It gamifies learning Math and Vietnamese through a Plants vs Zombies-inspired experience.

## Core Features

- Multiple game modes: Classic (V1), Adventure (V2), Local 2-Player Duel (V3), Online Duel via WebSocket (V4)
- Multiple-choice questions with 4 options (A/B/C/D) across subjects (math, vietnamese) and difficulties (easy, medium, hard)
- Timed exams with grading (A+ through F)
- Player profiles tracking stars, combos, session history
- Admin dashboard for managing questions, viewing player weaknesses, and creating exams
- Vietnamese language UI throughout

## Target Audience

Young children (grade 2, ~7-8 years old) in Vietnam. UI should remain simple, fun, and accessible.
